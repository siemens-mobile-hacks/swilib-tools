# Swilib Tools

```
swilib-tools.js [command]

Commands:
  swilib-tools.js update-cache             Update all caches.
  swilib-tools.js check <platform> <file>  Check swilib.vkp for errors.

Options:
  --help     Show help                                                 [boolean]
  --version  Show version number                                       [boolean]
```
